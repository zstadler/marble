Israel Hiking map (via osm.org.il)

Info

This map was generated from Open Street Map (OSM) data where it can be viewed and edited. All you need to do is create an account and start mapping. Note that the changes will not affect this map instantly. Below are some links related to this project

  • See our map on Github for instructions on how to create this map by yourself and links to map downloads.
  • See our site on Github in order to submit a change request an idea or an issue you encountered, this will help us improve your experience.
  • See Israel OSM Wiki Project for common rules of the Israeli Open Street Map community.
  • See Israel OSM Forum for the Israeli Open Street Map forum where you can post questions and look for answers.

usage

Due to technical limitations only level 7 zoom and up is supported, hence to use you will need to zoom at Israel location zoom to level 7.

This can be done by:

  • searching for Israel in the search option and clicking on the dot.
  • manually going to 34.882 and 31.69.
